# Bryce Vancuren - Portfolio Website

A modern, responsive portfolio website showcasing Bryce's journey from Computer Information Systems to teaching English, and now to AI development.

## Features

- **Responsive Design**: Works seamlessly on all devices (mobile, tablet, desktop)
- **Modern UI**: Clean and professional design with smooth animations
- **Contact Form**: Allows visitors to send messages directly from the website
- **Project Showcase**: Highlights key projects with relevant technology tags
- **Skills Section**: Organized display of technical and soft skills
- **Mobile Navigation**: Hamburger menu for better mobile experience

## Technologies Used

- HTML5
- CSS3 (with CSS Variables, Flexbox, and Grid)
- JavaScript (Vanilla JS with no external libraries)
- Responsive Design principles
- CSS Animations

## Page Structure

1. **Header**: Introduction with name and professional title
2. **About**: Brief professional journey description
3. **Projects**: Showcase of key professional projects
4. **Skills**: Technical and soft skills organized by category
5. **Contact**: Form for visitors to reach out and social media links
6. **Footer**: Copyright information

## Setup Instructions

1. Clone this repository or download the ZIP file
2. Open `index.html` in your browser to view the website
3. To make changes, edit the HTML, CSS or JavaScript files as needed

## Customization

- Colors: Edit the CSS variables in the `:root` selector in `styles.css`
- Content: Update the project information and skills in `index.html`
- Images: Replace or add images in an `images` folder (if needed)
- Form handling: The contact form uses frontend validation, but needs backend processing for actual message delivery

## Browser Compatibility

- Firefox (latest)
- Safari (latest)
- Edge (latest)
